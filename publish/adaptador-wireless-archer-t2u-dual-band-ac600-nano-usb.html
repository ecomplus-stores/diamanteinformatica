# Descrição do produto

Adaptador USB Wireless Nano AC600
- WiFi de alta velocidade
- Dual band Wireless
- Design Nano
- Segurança avançada

Dual Band de Alta Velocidade para Melhores Conexões
O Archer T2U Nano recebe sinais Wi-Fi em duas bandas separadas. O suporte à tecnologia 256QAM aumenta a taxa de transferência de dados de 2.4 GHz com 150 Mbps para 200Mbps para um desempenho 33% mais rápido. 433Mbps em 5GHz é ideal para streaming de vídeo HD e jogos online sem atraso, enquanto 200 Mbps em 2.4GHz é perfeito para uso cotidiano, como navegação na Web com dispositivos legados.

Design miniatura - discreto
Com tamanho miniatura e design elegante, os usuários podem inserir o adaptador nano em uma porta USB sem a necessidade de remover, mesmo ao guardar o notebook em uma mochila.

Ideal para viagens e com Wi-Fi roaming
Os crescentes serviços públicos de Wi-Fi vêm com um inconveniente - você pode ter que selecionar e se conectar repetidamente a diferentes redes Wi-Fi durante a movimentação.

O Archer T2U Nano suporta Wi-Fi roaming que ajuda seus dispositivos a se conectarem ao Wi-Fi mais adequado automaticamente enquanto você se move, garantindo que sua rede não seja interrompida e, com seu tamanho, o Archer T2U Nano é sem dúvida uma ótima opção para viajantes.

Fácil configuração
Uma interface de usuário amigável para instalação simples.

Segurança avançada
Os padrões de criptografia WPA/WPA2 garantem que sua conexão sem fio esteja protegida contra intrusos

Sistema operacional
Suporta sistemas operacionais incluindo Windows, Mac OS.
